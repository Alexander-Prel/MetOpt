# Решение задачи линейного программирования симплекс-методом

## Общие сведения

**Ф.И.О.:** __________________________  
**Поток:** __________________________  
**Вариант:** ________________________  

В рамках данной работы была рассмотрена задача линейного программирования (ЗЛП), выполнено ее приведение к каноническому виду и реализовано программное решение с использованием двухфазного симплекс-метода на языке Python. Также результат был проверен с помощью надстройки **MS Excel «Поиск решения»**.

---

## Постановка задачи

Требуется минимизировать целевую функцию:

\[
Z = 3x_1 + x_2 + 4x_3 + 2x_4
\]

при ограничениях:

\[
\begin{cases}
2x_1 + x_2 + x_3 \le 8 \\
x_1 + x_3 + x_4 = 5 \\
x_2 + x_4 \ge 4 \\
x_1, x_2, x_3, x_4 \ge 0
\end{cases}
\]

---

## Приведение задачи к каноническому виду

Для приведения ЗЛП к каноническому виду были выполнены следующие преобразования:

1. Для ограничения вида `≤` добавлена переменная-слэк:
\[
2x_1 + x_2 + x_3 + s_1 = 8
\]

2. Для ограничения вида `=` добавлена искусственная переменная:
\[
x_1 + x_3 + x_4 + a_2 = 5
\]

3. Для ограничения вида `≥` вычтена переменная-излишек и добавлена искусственная переменная:
\[
x_2 + x_4 - s_3 + a_3 = 4
\]

Все переменные считаются неотрицательными.

---

## Формирование вспомогательной задачи (Фаза 1)

Так как в системе присутствуют искусственные переменные, была сформирована вспомогательная задача:

\[
w = a_2 + a_3 \rightarrow \min
\]

Целью первой фазы является поиск допустимого базисного решения исходной задачи.

---

## Решение вспомогательной задачи

Вспомогательная задача была решена симплекс-методом.  
В результате получено:

\[
w^* = 0
\]

Это означает, что допустимая область исходной задачи **не пуста**, и можно перейти ко второй фазе.

---

## Переход к основной задаче (Фаза 2)

После завершения первой фазы искусственные переменные были удалены из симплекс-таблицы.  
Далее была решена исходная задача минимизации целевой функции.

---

## Результат решения основной задачи

В результате применения симплекс-метода получено оптимальное решение:

- Оптимальная точка:
\[
x_1 = 0,\quad x_2 = 0,\quad x_3 = 0,\quad x_4 = 5
\]

- Значение целевой функции:
\[
Z^* = 10
\]

---

## Проверка решения в MS Excel

Для проверки корректности результата задача была решена с помощью надстройки **MS Excel «Поиск решения»**.

Параметры Solver:
- Целевая ячейка — значение функции Z (минимизация)
- Изменяемые ячейки — \(x_1, x_2, x_3, x_4\)
- Метод решения — **Simplex LP**
- Ограничения заданы в соответствии с условием задачи
- Переменные заданы как неотрицательные

Результат Excel полностью совпал с результатом, полученным программно:
\[
Z^* = 10
\]
